# Security Policy

## Reporting a Vulnerability

Found a security issue? Please email me directly at **moasilturk@gmail.com**, instead of opening a public issue.

Include whatever you can:
- What's the problem?
- How can I reproduce it?
- Which version are you using?

I'll try to get back to you as soon as possible and work on a fix.

## What Counts as a Security Issue?

- Bugs that could make CTGuard miss real vulnerabilities
- Crashes or DoS with crafted input
- Anything that breaks the analysis integrity

For regular bugs or feature requests, just open a GitHub issue.

## Thanks

If you report a valid security issue, I'll happily credit you in the release notes (unless you prefer to stay anonymous).
